# 🦕 Bedrock Bass-Ax - Flintstones Rock Plugin

**Yabba Dabba Doo!** A silly dinosaur bass + caveman rock guitar instrument made for kids by Laila, Lilibet Diana, and Dad using Amorph!

Perfect for jamming with the Moog and making Bedrock Band songs.

### 🎛️ The Knobs (Twist them like crazy!)

- **Roar Knob** (Dino Growl)  
  Turns the sound from a mumble into a **"YIP-YIP-YIP!"** dinosaur bark!

- **Yabba Knob** (Fred Crunch)  
  Adds the **grit**. Crank it up to make the rocks crumble!

- **Rock Club Knob** (Guitar Bite)  
  Makes it sound less like a boring synth and more like a real **caveman lead guitar**!

### How to Use
1. Copy the code from `bedrock-bass-ax.cmajor`
2. Paste it into **Amorph**
3. Hit **Compile**
4. Play low notes = Dino roaring bass  
   Play higher notes + twist the knobs = Fred Flintstone rock guitar!
5. Layer it with your Moog for full **Flintstones Rock Band** songs!
